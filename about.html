About the studio

We started with one mop.

Founded 2019
Agadir · Souss-Massa

Digi Moqef is a small cleaning studio that thinks about cleaning the way a film studio thinks about a scene — light, composition, what's in the frame, what isn't.

We started with one mop and a Saturday morning. Word travelled. Five years later we run overnight crews for offices, restaurants, clinics, and a few of Agadir's nicer apartment buildings.

We answer the phone. We send the same crew. We don't subcontract. If something breaks, we replace it. If we miss a spot, we come back the same day, no charge.
